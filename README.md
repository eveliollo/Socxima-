# SOCXIMA.IA — Red Autónoma de Orden y Confianza

> La infraestructura que transforma cualquier conexión en una red segura, ordenada y verificable, anclada a Solana.

## ✅ Lo que este repositorio incluye
- Servidor público y página principal del buscador
- Estructura de conexión y módulos de acceso
- Interfaz pública y documentación de uso
- **Versión de referencia/demostración del núcleo**: funciona para pruebas, pero no contiene la lógica de sincronía y mezcla exclusiva de producción.

## 🔒 Lo que NO está aquí
El sistema de orden de 4 ejes completo, semillas internas, constantes de mezcla y lógica de confianza fina se mantienen en repositorio privado exclusivo: son la tecnología diferenciadora de SOCXIMA.

## 🚀 Cómo probar
Lee las instrucciones de instalación y ejecución en este repositorio; para uso en producción se requiere integración con el núcleo privado autorizado.
